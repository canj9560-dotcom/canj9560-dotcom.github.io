# 纪灿的博客

## 博客简介
本博客基于Hexo框架搭建，采用Fluid主题实现简洁的Material-Design风格视觉体验，是个人技术学习、生活技巧的分享平台。
- 作者：纪灿（学号：202215004676）
- 创建时间：2025年12月8日
- 核心功能：文章发布、分类/标签管理、归档查询、独立「关于」页面
- 内容方向：技术笔记、低成本生活技巧（如居家收纳）等
- 主题特性：响应式布局，导航栏包含首页、归档、分类、标签、关于模块
